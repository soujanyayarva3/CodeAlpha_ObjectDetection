# CodeAlpha Object Detection

This project is a real-time face detection system developed using Python and OpenCV as part of the CodeAlpha Artificial Intelligence Internship.

## Features

* Real-time webcam detection
* Detects human faces
* Draws bounding boxes around detected faces

## Technologies Used

* Python
* OpenCV

## Installation

```bash
python -m pip install opencv-python
```

## How to Run

```bash
python object_detection.py
```

## Controls

* Press `q` to quit the webcam window
